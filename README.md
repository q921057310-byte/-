# 机械设计手册 发布目录

这个目录用于整理当前可发布版本，方便后续上传到 GitHub。

## 目录说明

- `html/output/`
  - HTML 版完整文件
  - 可直接用浏览器打开 `index.html`，或配合本地静态服务器使用

- `apk/机械设计手册.apk`
  - 安卓安装包

- `chm/机械设计手册.chm`
  - 电脑端 CHM 版本

- `tools/`
  - 构建和辅助文件
  - 包含启动器、打包脚本、HTML 生成脚本等

## 备注

- `机械设计手册.exe` 是电脑端启动器，不是单文件完整内容包，需配合 HTML 资源使用。
- `机械设计手册-电脑版.zip` 是电脑端完整分发压缩包。
